创建github账号

正文内容（替换成自己的）

基于hexo已经安装好的步骤

1.注册github

首相你要在github上注册一个账号，

步骤1：填写你的用户名，Email，密码。

步骤2：进入Step1，填写用户名，Email,密码。后面会有验证，会通过qq邮箱发送给你，进行验证。

步骤3：进入Step2，选择仓库。

步骤4：进入Step3，让你做一些选择，完善一些信息。也可以直接跳过步骤，这样你的账号就注册好了。

2.创建仓库

注册完上面步骤后登录，进行以下创建仓库步骤：

步骤1：点击new repository新

这样你的仓库就创建好了。
